-------


### 导入
> * 手动导入：
>
>> * **SwpBanner** 文件夹导入项目中。
>> * **`#import "SwpBannerHeader.h"`**
>> * 依赖库 **SDWebImage** , 需要手动导入。
>>
>> * **[SDWebImage](https://github.com/rs/SDWebImage)**


> -------

> * CocoaPods 导入:
>
>> * **pod search SwpBanner**
>> * **pod 'SwpBanner'**
>> * **`#import <SwpBanner/SwpBannerHeader.h>`**
>> * **pod** 导入会自动添加依赖库, 无需手动导入
